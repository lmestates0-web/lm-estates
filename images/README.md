# Hero Images Directory

Place your hero image here as hero.jpg.

Recommended dimensions: 1600px width or larger for best quality.
The image will be displayed as a responsive background covering the hero section.

## How to add your image:
1. Upload your hero image to this directory
2. Name it hero.jpg
3. The website will automatically use this local image

## Image optimization tips:
- Use JPEG format for photos
- Aim for file size under 500KB for fast loading
- Minimum recommended width: 1200px
- Aspect ratio: landscape (16:9 or similar)
